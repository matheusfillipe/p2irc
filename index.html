command | curl -d @- p2ir.cf/irc.server/room


For multiline (files, etc)

cat file | curl --data-binary @- p2ir.cf/ro



Do not put # on the room name

Use '-' to send a private message e.g.

echo "hey greg" | curl --data-binary @- p2ir.cf/isr.server.com/-greg https://github.com/matheusfillipe/p2irc
